Step 1: Open VS Code and the Folder
Since you already have java_folder open in VS Code, move to the next step.

Step 2: Open the Terminal
Open the VS Code Terminal by pressing Ctrl + ` (backtick) or going to View > Terminal.

Step 3: Initialize a New Git Repository
